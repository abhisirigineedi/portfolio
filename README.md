# Personal Portfolio Website

## 1. Project Overview
This project is a clean, modern, and mobile-responsive personal portfolio website for **Sirigineedi Abhi Sai Ram**. It showcases his professional profile as a B.Tech CSE (Cyber Security) graduate, including his skills, projects, education, certifications, and other credentials. It is designed with a dark-theme glassmorphism aesthetic to be visually appealing and user-friendly.

It is built entirely using standard web technologies: HTML5, CSS3, and JavaScript, ensuring it is beginner-friendly and requires no complex build tools or dependencies.

## 2. Website Sections
The website structure is divided into the following logical sections:

- **Hero / Home:** An introductory section featuring a prominent greeting, name, tagline, short bio, and quick-action buttons (View Projects, Contact Me).
- **About:** A professional summary highlighting education, core tech interests, and career goals.
- **Skills:** A categorized display of technical and soft skills:
  - **Programming:** C, C++, Python
  - **Databases:** SQL, PL/SQL
  - **Tools & Platforms:** Visual Studio Code, Git, GitHub, Oracle Database, Jira
  - **Soft Skills:** Problem Solving, Attention to Detail, Adaptability, Communication, Continuous Learning
- **Education:** A timeline of academic history from high school to B.Tech.
- **Projects:** Interactive cards detailing significant projects (AI DDoS Detection, TechMate Assistant, AutoMessenger) along with technologies used and links to source code.
- **Certifications:** Professional certifications from NPTEL and SSSIT with "View Certificate" links to Google Drive.
- **Extracurricular Activities:** Details of NSS community service and volunteering experience with a "View Certificate" link.
- **Contact:** Contact information and a working contact form powered by [Web3Forms](https://web3forms.com/) that sends messages directly to your email.
- **Footer:** Copyright information and social links (LinkedIn, GitHub).

## 3. Contact Form Setup
The contact form uses **Web3Forms** to deliver messages to your email inbox. To configure it:

1. Go to [https://web3forms.com/](https://web3forms.com/)
2. Enter your email address and get a free Access Key
3. Open `index.html` and replace the `access_key` value with your key
4. The form will now send messages directly to your email

## 4. How to Deploy Using GitHub Pages
Because this site consists purely of static files, it is perfectly suited for free hosting via GitHub Pages. Follow these steps:

1. **Create a new repository on GitHub:**
   Go to GitHub, log in, click the "+" icon in the top right, and select "New repository". Name it (e.g., `portfolio`). Leave it public.

2. **Upload all project files:**
   Push your files (including `index.html`, `style.css`, `script.js` and the `assets` folder) to the root of your newly created repository via Git in the terminal, or upload them directly in the GitHub web interface.

3. **Go to repository settings:**
   In your repository view on GitHub, click the **Settings** tab.

4. **Enable GitHub Pages:**
   In the left sidebar of the Settings page, scroll down and click on **Pages**.

5. **Select the main branch:**
   Under the "Build and deployment" section, choose "Deploy from a branch". In the Branch dropdown menu, select `main` (or `master`) and click **Save**.

6. **Access your live portfolio website:**
   After a minute or two, GitHub will publish the site. A link to your live portfolio will appear at the top of the Pages settings menu (e.g., `https://[your-username].github.io/portfolio/`).
